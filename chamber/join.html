Applicant Information

NP Membership (Non-Profit)

  • No membership fee
  • Access to networking events specifically for non-profit organizations
  • Listing in the directory of non-profit members on the website

Bronze Membership

  • Basic listing in the member directory
  • Access to one educational webinar per quarter
  • Discounted rates for advertising opportunities on the website

Silver Membership

  • Enhanced listing in the member directory with a logo
  • Access to two educational webinars per quarter
  • Priority consideration for speaking opportunities at events
  • 10% discount on event registrations

Gold Membership

  • Featured listing in the member directory with a prominent logo and description
  • Access to all educational webinars
  • Priority sponsorship opportunities for events
  • 20% discount on event registrations
  • Exclusive access to networking events with industry leaders
